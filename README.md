# cn_timer_bot
(WIP)
A clojure telegram timing bot 

## Usage
make a bot with bot father
add a file `bot-token.txt` with your token.

commands:
```
/info (get target time)
/until (get time until target time)
/set (set target time DD.MM.YYYY HH:MM)
```
